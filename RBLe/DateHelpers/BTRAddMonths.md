# BTRAddMonths Function

Returns a target date after adding specified years.

## Syntax

```excel
=BTRAddMonths(target, months)
```

Parameter | Type | Default | Description
---|---|---|---
`target` | DateTime |  | The target date.
`months` | Int32 |  | Number of months to add.
