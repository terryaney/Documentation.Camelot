# BTRAddYears Function

Returns a target date after adding specified years.

## Syntax

```excel
=BTRAddYears(target, years)
```

Parameter | Type | Default | Description
---|---|---|---
`target` | DateTime | `` | The target date.
`years` | Int32 | `` | Number of years to add.
